# EasyBroker API Consumer

## Description
This Ruby script consumes the EasyBroker API to read all properties from a staging account and print their titles. It includes a unit test using RSpec to ensure functionality.

## Prerequisites
Make sure you have the following versions of rbenv and Ruby installed:
- rbenv 1.2.0
- Ruby 3.2.1

## Setup the API key and check package version

You can check your version by running:
```bash
rbenv -v
ruby -v
